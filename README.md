 ## About the Project
 -본 프로젝트는 거리가 떨어져 있는 여러 사람이 한 곳에 모일 경우에 특정한 누군가가 불이익을 받지 않고 공평하게 장소를 정할 수 있도록 도와줍니다.
 이를 위해 중간지점에 해당하는 위치를 카카오맵을 활용하여 제공합니다.
 
 ## 기술스택

 - AWS EC2
